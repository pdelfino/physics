# Mathematical Modelling of Physical Phenomena

![A Philosopher Lecturing on the Orrery](https://upload.wikimedia.org/wikipedia/commons/thumb/d/d3/Wright_of_Derby%2C_The_Orrery.jpg/960px-Wright_of_Derby%2C_The_Orrery.jpg)

*"A Philosopher Lecturing on the Orrery" (1766) by Joseph Wright of Derby — [Wikipedia](https://en.wikipedia.org/wiki/A_Philosopher_Lecturing_on_the_Orrery)*

Physics simulations built with Processing, modelling real-world systems with accurate proportions and dynamics.

## About

Coursework for **Modelagem Matematica 2** (Mathematical Modelling 2) at EMAp/FGV, taught by Professor Paulo Cezar P. Carvalho, PhD (2018.2).

Two projects that translate physical systems into interactive simulations, each built to respect real-world proportions and measurements.

## Projects

### Pong -- Table Tennis Simulation

A from-scratch Pong game modelled after real table tennis. Official dimensions and proportions are preserved:

- Table: 274 cm x 152.5 cm
- Ball: 40 mm diameter
- Racket: 17 cm height
- Official color scheme: blue table, white lines, white ball, red rackets

![Table tennis dimensions](https://github.com/pdelfino/physics/blob/master/rules-sizes.png)

![Pong gameplay](https://media.giphy.com/media/2440Yhc1g5va07Ly09/giphy.gif)

**Controls:** Arrow keys move the rackets (UP/DOWN for left player, LEFT/RIGHT for right player).

### Solar Eclipse Simulation

A simulation of the Sun-Earth-Moon system modelling a solar eclipse event. The simulation attempts to maintain proportional sizes and distances between celestial bodies.

![Moon-Earth-Sun proportions](https://github.com/pdelfino/physics/blob/master/proportion-moon-earth-sun.png)

![Eclipse geometry](https://github.com/pdelfino/physics/blob/master/solar-eclipse-diff.png)

![Solar eclipse simulation](https://media.giphy.com/media/ZvUMWYKM2RQC6DMeDx/giphy.gif)

## How to Run

1. Install [Processing](https://processing.org/download).
2. Open the `.pde` file for the desired project.
3. Click the Run button in the Processing IDE.
